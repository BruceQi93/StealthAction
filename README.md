# StealthAction
a third person escape game

### introduce

这是根据Unity官网上的案例做出来的，主角需要秘密地通过重重机关，来躲避巡逻机器人，一旦被他们发现将很难脱身，游戏中有一些机关，钥匙，主角需要通过机关拿到钥匙，才能坐电梯离开。
